# Gen_Verif_Hash

# Hash Generator and Verifier

A simple Python tool with a graphical interface for generating and verifying password hashes (Argon2, bcrypt) and file hashes (SHA-512).

---

## Features

- Secure hash generation for passwords with Argon2 and bcrypt
- Hash generation for files with SHA-512 (and other algorithms if added)
- Password and file verification via hash
- Intuitive graphical interface with dedicated tabs
